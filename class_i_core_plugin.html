Detailed Description

Interface for plugin CorePlugin.

This interface represents most significant plugin in whole system. If plugin implements this interface it should take care of whole system plugins interactions. When PluginLoader class loads plugins from directory it searchs the plugin which inherits from ICorePlugin. If it find one - it will send all loaded plugins to it by using AddPlugin() method.
